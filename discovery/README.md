# 发现工具（Discovery Tools）

这个目录存放辅助发现新 spam 账号的工具。

## 推荐策略（2026 年现实版）

1. **最有效**：用户通过油猴脚本的「举报」按钮实时贡献
2. **次有效**：维护者手动在 X 上搜索关键词 + 观察模式
3. **辅助**：有 X API 配额的人可以运行搜索脚本做 seeding（成本较高，不推荐作为主力）

## 未来可能工具

- `seed-from-search.js`：使用 X API v2 recent search 按关键词找候选账号（需要付费 credits）
- 简单的网页版提交表单（Cloudflare Worker）

目前阶段，优先把用户脚本 + GitHub blocklist 做好，数据质量比数量更重要。

## 关键词建议（给人工搜索用）

- "寻固炮" -is:retweet min_faves:0
- "🌸 寻固炮 🌸"
- "固炮" "点击主页"
- 配合 "动漫头像 + 低互动" 人工判断

## 账号判断标准（严格）

必须同时满足多条才考虑收录：
- 使用固定模板文案
- 动漫/美女头像为主
- 几乎无真实互动
- bio 或多条推文包含引流话术
- 明显为引流/诈骗目的
