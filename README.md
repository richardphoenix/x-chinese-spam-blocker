# X 中文 Spam 防御工具

**GitHub**: https://github.com/richardphoenix/x-chinese-spam-blocker

针对中文 X（原 Twitter）上大规模 “寻固炮” 等低俗/诈骗 spam 的社区防御方案。

针对中文 X（原 Twitter）上大规模 “寻固炮” 等低俗/诈骗 spam 的社区防御方案。

## 现状问题

大量机器人账号使用动漫头像 + 固定模板（🌸 寻固炮 🌸 点击主页）刷屏，严重破坏中文讨论环境。这些账号主要目的是引流到微信/Telegram 进行诈骗或卖服务。

## 推荐方案（已验证有效）

**GitHub 托管黑名单 + 高质量 Tampermonkey 油猴脚本**

这是 2025-2026 年社区里最成功、最可持续的模式（参考 X.com Chain Blocker、Twitter Block With Love 等成熟项目）。

### 为什么不主要靠 X API 大规模抓账号？

- X API v2 已经是 pay-per-use，搜索成本高
- 搜索结果已经被平台 spam filter 严重过滤
- 账号发现最有效的方式是 **社区举报 + 启发式规则**，而不是纯 API 爬取

## 核心功能（脚本将提供）

- **时间线实时隐藏**：自动隐藏匹配黑名单或关键词的推文/账号（支持「寻固炮」「小狗求抱抱 / 免费曰p」和「想找会疼人的哥哥」等变种）（最推荐用法）
- **从维护者黑名单批量拉黑**：只对 Richard 审核确认过的账号开放批量拉黑（10秒/个，带队列、暂停、取消）
- **社区提交机制**：任何用户都可以一键提交疑似 spam 账号到 GitHub Issue，由维护者审核后才进入正式黑名单
- **误杀防护**：只有经过人工审核的账号才会进入可拉黑黑名单 + 支持本地白名单

## 黑名单治理模式

- 正式黑名单（`blocklist.json`）由维护者人工审核维护
- 社区用户可通过脚本提交账号（进入待审核状态）
- 批量拉黑功能**只针对正式黑名单**生效，最大程度降低误杀风险
